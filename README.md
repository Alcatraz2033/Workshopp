## Workshopp

### Descargar el workshopp 1
```git
git clone https://github.com/Alcatraz2033/Workshopp.git
cd Workshopp
gcc workshop1.c -o workshopp (linux)
gcc workshop1.c -o workshopp.exe (windows)
```
